Privacy Policy for PDF Librarian

Last updated: 2026-03-11

English

PDF Librarian is a macOS app that helps users organize book and academic paper PDFs.

Data We Access

The app only accesses files and folders that the user explicitly selects through the standard macOS file picker.

The app may read information from those selected PDFs in order to identify possible title, ISBN, DOI, or other metadata hints; search public metadata sources requested by the user; write confirmed metadata back into the selected PDF; and rename the selected PDF after user confirmation.

Network Requests

When the user chooses to search for metadata, the app may send query terms such as a title, ISBN, DOI, or similar metadata hints to public metadata sources, including Google Books, Open Library, Douban web search, and the Library of Congress. These requests are initiated only when the user performs a metadata search.

Data Collection

PDF Librarian does not require a user account and does not use advertising SDKs, tracking SDKs, or third-party analytics SDKs. PDF Librarian does not intentionally collect personal profile data such as name, email address, phone number, precise location, contacts, or payment information.

Local File Processing

Metadata writing and file renaming are performed on the user-selected files. The app does not upload the full PDF document to the developer's server.

Contact

For privacy questions, visit the support page or open an issue at GitHub Issues.

简体中文

PDF Librarian 是一款用于整理书籍和学术文献 PDF 的 macOS 应用。

我们会访问哪些数据

应用只会访问用户通过 macOS 系统文件选择器明确选中的文件和文件夹。

应用可能会读取这些已选 PDF 中的部分信息，用于识别标题、ISBN、DOI 等元数据线索；在用户发起操作后查询公开元数据来源；将用户确认后的元数据写回所选 PDF；以及在用户确认后对所选 PDF 进行重命名。

联网请求

当用户主动执行元数据搜索时，应用可能会将标题、ISBN、DOI 或类似元数据线索发送给公开元数据来源，包括 Google Books、Open Library、豆瓣网页检索和 Library of Congress。这些联网请求只会在用户主动触发搜索时发起。

数据收集说明

PDF Librarian 不要求用户注册或登录，不使用广告 SDK、跟踪 SDK 或第三方分析 SDK，也不会主动收集姓名、邮箱、手机号、精确位置、通讯录或支付信息等个人资料。

本地文件处理

元数据写入和文件重命名都针对用户选中的本地文件执行。应用不会把完整 PDF 文件上传到开发者服务器。

联系方式

如有隐私相关问题，请访问 支持页面，或通过 GitHub Issues 联系。
