PDF Librarian Support

If you need help with PDF Librarian, please open an issue at GitHub Issues.

English

When contacting support, please include:

  • macOS version
  • App version
  • Whether the issue happens on one PDF or many PDFs
  • Which metadata source was enabled
  • A short description of the problem

Common support topics include metadata lookup failures, metadata write failures, rename mismatches, packaging or launch issues, and language or UI display issues.

简体中文

如果你需要获取 PDF Librarian 的支持，请通过 GitHub Issues 提交问题。

建议一并提供以下信息：

  • macOS 版本
  • 应用版本号
  • 问题是出现在单个 PDF 还是多个 PDF
  • 当时启用了哪些元数据来源
  • 问题的简短描述

常见支持场景包括搜索不到元数据、元数据写入失败、重命名结果不符合预期、打包或启动问题，以及语言或界面显示问题。
